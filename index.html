// 02 — Projets

Réalisations

Projet 01

Infrastructure AD — Groupe GSB

Déploiement complet d'une infrastructure Active Directory multi-sites pour un groupe pharmaceutique fictif. Includes : contrôleurs de domaine redondants, GPO de sécurité, DFS pour partage de fichiers, DNS/DHCP centralisé. Topologie réseau modélisée sous GNS3.

Windows Server Active Directory GPO DFS GNS3
Terminé
Projet 02

Supervision réseau avec Zabbix

Mise en place d'une solution de supervision complète : déploiement Zabbix, configuration d'agents SNMP sur équipements réseau, tableaux de bord personnalisés, alertes par mail. Surveillance de la disponibilité et des performances des services critiques.

Zabbix SNMP Linux Bash
Terminé
Projet 03

Parc informatique GLPI + FusionInventory

Déploiement de GLPI avec le plugin FusionInventory pour l'inventaire automatique du parc. Configuration des tickets, gestion des actifs, liaison avec l'annuaire LDAP.

GLPI FusionInventory LDAP MySQL
Terminé
Projet 04

Segmentation réseau — VLAN & DMZ

Architecture réseau segmentée avec VLAN (administration, production, invités), mise en place d'une DMZ pour exposition sécurisée de services web, règles de pare-feu Stormshield et routage OSPF inter-VLAN.

VLAN OSPF DMZ Stormshield GNS3
Terminé
